# ModernCard — MIT App Inventor Extension

Modern card UI with gradients, rounded corners, and shadows.

## How to build the .aix file

1. Go to the **Actions** tab above
2. Click **Build ModernCard Extension** on the left
3. Click **Run workflow** → **Run workflow**
4. Wait ~5 minutes for it to finish
5. Click the finished run → scroll down to **Artifacts** → download **ModernCard-extension**
6. Unzip it — you'll find the `.aix` file inside

## How to use in App Inventor

1. Open [ai2.appinventor.mit.edu](https://ai2.appinventor.mit.edu)
2. Extensions panel → Import extension → upload the `.aix`
3. Drag ModernCard into your project and use the blocks!
